# Instagram 클론코딩

## 📌 프로젝트 설명

### 🗓️ 개발기간
> 2024/06/10 ~ 2024/06/19

### 💁🏻‍♀️ 팀 구성
> <b>김시연</b> : 프론트엔드, 디자인(클론디자인)

### 💬 작품을 제작한 이유
> 1학년 때 했던 인스타그램 클론 코딩을 떠올리면 재미있었던 기억이 납니다. 그 프로젝트를 다시 보며, 제 자신이 얼마나 성장했는지를 확인하고 싶었기 때문에 인스타그램 클론코딩을 하게 되었습니다.

### 📹 시연영상

### ⭐️ 기능설명
#### 1. 로그인 페이지
<img width="1710" alt="스크린샷 2024-07-11 오전 9 17 58" src="https://github.com/kimsiyeon0223/Instagram-clone/assets/128461588/a3b9ee47-1ead-4973-88b1-99647b53645a">
- 입력하지 않은 입력칸이 있다면 alert창으로 입력하지 않은 칸을 알려줍니다. <br />
- username와 password를 입력하고 login 버튼을 누르면 username으로 로그인됩니다.<br />

#### 2. 회원가입 페이지
<img width="1710" alt="스크린샷 2024-07-11 오전 9 28 28" src="https://github.com/kimsiyeon0223/Instagram-clone/assets/128461588/61d0cc58-8739-48b7-b333-19ea0627ce30">
- 입력하지 않은 입력칸이 있다면 alert창으로 입력하지 않은 칸을 알려줍니다. <br />
- 입력칸에 값들을 입력하고 signup 버튼을 누르면 '회원가입이 완료되었습니다.'라는 alert창이 뜨고 로그인 페이지로 이동합니다.<br />
- Login 버튼을 누르면 로그인 페이지로 이동합니다.<br />

#### 3. 메인 페이지
<img width="1710" alt="스크린샷 2024-07-11 오후 12 07 42" src="https://github.com/kimsiyeon0223/Instagram-clone/assets/128461588/d6d478f3-88d3-4e12-9bff-894ae549c6a3">
- 모두보기 버튼 호버효과 -> 색깔 회색으로 바뀝니다.
- 팔로우 버튼 호버효과 -> 색깔 진한 파랑으로 바뀝니다.
- 하트를 누르면 좋아요 개수가 올라가고, 한번 더 누르면 취소할 수 있습니다.
